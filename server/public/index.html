Note App API

Welcome to the Note App API. This is the backend server for the Note App application.

Endpoints

GET /api/notes

Get all notes

GET /api/notes/:id

Get a single note by ID

POST /api/notes

Create a new note

PUT /api/notes/:id

Update a note

DELETE /api/notes/:id

Delete a note

Test Endpoints

GET /api/test

Simple API test endpoint

GET /api/cors-test

Test CORS configuration

Tools

CORS Test Tool - Use this to test API connectivity
